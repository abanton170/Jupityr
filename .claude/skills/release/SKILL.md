# Release Skill

## Description
Manages the release workflow for Jupityr, including:
- Version bumping
- Changelog generation
- Tag creation
- Release notes

## Usage
Prepare and execute a new release version.

## Steps
1. Determine the next version based on changes
2. Update version in pyproject.toml and setup.py
3. Generate changelog from commit history
4. Create git tag and release notes
