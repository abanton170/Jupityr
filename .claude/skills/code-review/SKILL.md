# Code Review Skill

## Description
Performs automated code review on changed files, checking for:
- Code quality and readability
- Potential bugs and edge cases
- Adherence to project conventions
- Security vulnerabilities

## Usage
Review staged or modified files and provide actionable feedback.

## Steps
1. Identify changed files using `git diff`
2. Read each changed file
3. Analyze for issues (style, bugs, security, performance)
4. Provide structured feedback with file paths and line numbers
