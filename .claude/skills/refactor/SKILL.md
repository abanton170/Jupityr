# Refactor Skill

## Description
Assists with code refactoring while maintaining correctness, including:
- Extracting functions and classes
- Renaming across the codebase
- Simplifying complex logic
- Improving code organization

## Usage
Identify refactoring opportunities and apply changes safely.

## Steps
1. Analyze the target code for refactoring opportunities
2. Propose changes with rationale
3. Apply refactoring edits
4. Run tests to verify correctness
